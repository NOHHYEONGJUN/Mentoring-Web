## 실습 시나리오

이 프로젝트는 다음 실습 시나리오를 지원합니다:

1. **시스템 배포**:
   - 실제 물리 서버에서 직접 시스템 배포

2. **도커를 이용한 컨테이너 배포**:
   - Dockerfile을 이용해 컨테이너를 빌드 및 배포

3. **웹 페이지 정보 수정**:
   - 웹 페이지의 사진, 내용 등 시각적으로 확인 가능한 내용을 수정
